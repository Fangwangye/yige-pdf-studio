## 一句话项目定位

本地运行的 Web 版 PDF 翻译工作台：左右双栏对照预览，接 OpenAI-compatible 模型，靠 **知识库** 保证术语与风格一致，并规划逐步引入 **MCP 检索** 与 **Claude Skill** 编排。

## 能力路线图（三期）

1. **一期 · 进行中**：结构化术语知识库（术语表 + 风格 + 禁译，按命中注入、可复用、导入导出）。
2. **二期**：MCP 检索接入（术语/风格库外置，翻译时实时检索注入，多系统共享）。
3. **三期**：Claude Skill 封装（把翻译流水线封装成 skill/agent，由 Claude Code 调度）。

详见 [功能说明/路线图.md](./功能说明/路线图.md)。
