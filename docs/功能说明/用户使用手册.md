# 用户使用手册

## 0. 启动

```powershell
python -m pip install -r requirements.txt
python -m uvicorn app.main:app --host 127.0.0.1 --port 8010
```

浏览器打开 <http://127.0.0.1:8010/>。

界面左侧为导航，右侧为工作区，共五个视图：**翻译工作台 / 知识库 / 任务与质检 / 接入设置 / 帮助**。

## 1. 接入设置（首次必做）

进入「接入设置」：

- **服务商**：选 `DeepSeek / OpenAI-compatible`；没有 API Key 想先看版式效果可选 `Mock 版式测试`（只复制原 PDF，不翻译）。
- **Base URL**：填到 `/v1` 这一层，例如 `https://api.deepseek.com/v1`。
- **模型**：例如 `deepseek-v4-pro`。
- **API Key**：`sk-...`。
- **版式引擎**：默认 `BabelDOC`（版式更好）；异常时可切 `pdf2zh legacy`。
- **质量模式**：质量优先（线程 1，最稳）/ 均衡 / 速度优先。
- **保留目录页 / 保留页码**：对目录页、版权页等不想翻译的页填页码（如 `2,4-5`）。

左下角的连接指示会显示当前是 Mock、未配置 Key 还是已连接。

## 2. 知识库（建议先配）

进入「知识库」：

- 顶部下拉（在工作台）可切换已有配置：学术论文 / 技术文档 / 商务合同。
- 编辑区填写三类规则：

```text
术语：
large language model = 大语言模型
scaling law = 缩放定律

风格：
使用正式、流畅的学术中文，保持论文语气。

禁译：
模型名、数据集名、公式、引用保留原文。
```

- 命名后点**保存**；可**导出**为 `*.knowledge.json` 备份，或**导入**他人配置。
- 知识库优先级高于自动抽取的文档上下文，但不会覆盖占位符/公式/引用保护规则。

## 3. 翻译

进入「翻译工作台」：

1. **选择 PDF**：选中后左栏立即预览原文。
2. 确认**源语言 / 目标语言**（如 `en` → `zh`）。
3. 选择**知识库**配置。
4. 点**开始翻译**。状态栏显示排队 → 处理中（含引擎日志）→ 完成。
5. 完成后右栏出现译文 PDF，顶部「下载译文 PDF」「打开译文」可用。

> 论文/长文档可能需要几分钟；首次运行还要初始化模型与字体。

## 4. 任务与质检

进入「任务与质检」：

- **任务历史**：点任意历史任务可重新载入其原文/译文预览与质检报告。
- **质量报告**：
  - 页面生成、占位符/乱码扫描、异常页自动回退、跨页连续性、知识库命中。
  - 标「查」的页建议人工复查；失败任务会显示引擎错误摘要。

## 常见问题

- **右侧没出译文**：确认任务状态为 `succeeded`；失败看状态栏错误。
- **术语没按知识库翻**：确认翻译前已在工作台选中对应知识库配置并已保存。
- **很慢**：首次初始化模型所致；或把质量模式调到「均衡/速度优先」。
- **乱码/占位符**：在质检报告查看具体页，必要时切 `legacy` 引擎重试。
