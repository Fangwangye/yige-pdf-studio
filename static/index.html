快速上手

  1. 接入设置：填 Base URL / 模型 / API Key（或选 Mock 仅测版式）。
  2. 知识库：选一套术语/风格规则，或新建保存自己的。
  3. 翻译工作台：选 PDF → 设语言 → 开始翻译，左右对照预览。
  4. 任务与质检：查看历史任务、占位符/乱码扫描与回退情况。
